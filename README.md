## Simple Interest Calculator

This project calculates simple interest using the formula:

Simple Interest = (P × R × T) / 100

### Inputs:
- P: Principal amount
- R: Rate of interest
- T: Time in years

### Example:
P = 1000, R = 5%, T = 2  
Simple Interest = 100

### How to Run:
Run the script:
./simple-interest.sh
